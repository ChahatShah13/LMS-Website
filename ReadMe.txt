# LMS Website

## Overview
This project is a Learning Management System (LMS) website designed to provide a platform for online courses. It includes features such as course listings, user authentication, and interactive content.

## Technologies Used
- React
- Tailwind CSS
- Google Fonts
- HTML
- CSS
- JavaScript

## File Structure
- `client/src/index.css`: Contains global styles and custom Tailwind CSS configurations.
- `client/src/components/Hero.jsx`: The Hero component for the homepage.
- `assets`: Directory for storing images and other assets.

## Setup Instructions
1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies using `npm install`.
4. Start the development server using `npm start`.

## Customization
- Fonts can be customized by modifying the `@import` statements in `index.css`.
- Tailwind CSS configurations can be adjusted in the `@theme` section of `index.css`.
- Components can be modified or added in the `client/src/components` directory.
